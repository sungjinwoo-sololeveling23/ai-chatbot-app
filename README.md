# AI Scholar 🎓\n\nA modern, feature-rich AI learning companion powered by Claude AI, built with React and Vite.\n\n## 🎯 Features\n\n- ✨ **Smart AI Responses** - Powered by Claude Sonnet\n- 🎤 **Voice Input** - Record and transcribe voice messages\n- 📸 **Camera & Image Upload** - Send images for analysis\n- 📐 **Math Rendering** - Beautiful LaTeX formula support\n- 🎨 **Customization** - Multiple personas, themes, and font sizes\n- 🔐 **Google Authentication** - Secure login with Google\n- ⚡ **Real-time Chat** - Smooth, responsive messaging\n\n## 🚀 Quick Start\n\n### Prerequisites\n- Node.js 16+ installed\n- Google Cloud credentials (for OAuth)\n- Anthropic API key (for Claude)\n\n### Installation\n\n```bash\ngit clone https://github.com/sungjinwoo-sololeveling23/ai-chatbot-app.git\ncd ai-chatbot-app\nnpm install\n```\n\n### Configuration\n\n1. **Set up Google OAuth:**\n   - Go to [Google Cloud Console](https://console.cloud.google.com)\n   - Create a new project\n   - Enable Google+ API\n   - Create OAuth 2.0 credentials\n   - Copy your Client ID\n   - Paste it in `src/App.jsx` at `GOOGLE_CLIENT_ID`\n\n2. **Set up Anthropic API:**\n   - Get your API key from [Anthropic Console](https://console.anthropic.com)\n   - Add it to your environment variables or use it directly in API calls\n\n### Development\n\n```bash\nnpm run dev\n```\n\nThe app will open at `http://localhost:3000`\n\n### Build for Production\n\n```bash\nnpm run build\n```\n\n## 📁 Project Structure\n\n```\n├── src/\n│   ├── App.jsx           # Main application component\n│   └── main.jsx          # React entry point\n├── index.html            # HTML template\n├── vite.config.js        # Vite configuration\n├── package.json          # Dependencies\n└── README.md             # This file\n```\n\n## 🔧 API Integration\n\n### Claude API\nThe app uses the Anthropic Claude API for:\n- Text responses to user queries\n- Audio transcription\n- Image analysis\n\n### Google OAuth\nSecure authentication via Google accounts\n\n## 🎭 Customization\n\n### Personas\nThree AI personas available:\n- **Aria** (✦) - Warm & friendly\n- **Sage** (🧠) - Deep thinker & analyst\n- **Spark** (⚡) - Creative & energetic\n\n### Themes\n- Violet, Cyan, Rose, Amber\n\n### Font Sizes\n- Small, Medium, Large\n\n## 🌐 Browser Support\n\n- Chrome/Edge 90+\n- Firefox 88+\n- Safari 14+\n\n## 💻 Technologies Used\n\n- **React 18** - UI framework\n- **Vite** - Build tool & dev server\n- **Claude API** - AI backend\n- **Google OAuth** - Authentication\n- **KaTeX** - Math rendering\n- **Web APIs** - Camera, Microphone, Canvas\n\n## 🌍 Deployment\n\nThe app can be deployed to any static hosting:\n- Vercel\n- Netlify\n- GitHub Pages\n- AWS S3 + CloudFront\n\n## 🐛 Troubleshooting\n\n### \"Camera access denied\"\n- Check browser permissions\n- Ensure HTTPS in production\n\n### \"Microphone permission denied\"\n- Check system audio permissions\n- Try a different browser\n\n### Google login not working\n- Verify Client ID is correct\n- Check that origin URL is whitelisted in Google Console\n\n### Claude API errors\n- Ensure API key is valid\n- Check rate limits\n- Verify network connectivity\n\n## 📄 License\n\nMIT\n\n## 💬 Support\n\nFor issues or questions, please open an issue on GitHub.\n
